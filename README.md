# Todo List App

Веб-приложение для управления списком задач с авторизацией и синхронизацией данных. Поддерживает гостевой режим, регистрацию пользователей, хранение задач в базе данных, а также сортировку и фильтрацию задач.


## Стек технологий

| **Фронтенд** | HTML, CSS, JavaScript, фреймворк Elmish |
| **Бэкенд** | Node.js, Fastify |
| **База данных** | SQLite |
| **Деплой** | Railway (бэкенд + фронтенд) |


## Как запустить локально

### 1. Клонировать репозиторий
```bash
git clone https://github.com/Palaye-Royale/javascript-todo-list-tutorial.git
cd javascript-todo-list-tutorial
```

### 2. Установить зависимости для бэкенда
```bash
cd backend
npm install
```

### 3. Запустить сервер
```bash
npm start
```
Сервер будет доступен по адресу http://localhost:3000
